h2. Cloves Carneiro Jr

Cloves Carneiro Jr. is a software engineer and web application developer with over twelve years of experience creating web applications for companies in many fields, including startups, and telecommunication and financial companies. He has been using Ruby on Rails since its early days, being a full-time Rails developer for 4 years, he currently works for Unspace Interactive in Toronto. h2. Rida Al Barazi

Rida Al Barazi is a passionate web developer experienced in building smart web applications for startups. He has been designing and building for the web since 2002. He started working with Rails in 2005 and spoke at different web and Rails conferences in Europe and the Middle East.
